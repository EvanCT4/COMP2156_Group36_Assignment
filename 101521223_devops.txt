In this course, students will learn the procedures, protocols, and best practices that a DevOps engineer brings to a workplace environment to increase productivity. They will analyze and evaluate various developer operation implementations and gain hands-on experience using industry-standard toolkits. Students will create solutions to complex business problems by adhering to the core responsibilities of DevOps.

Students will also learn about data formats and types, including:
- Extensible Markup Language (XML)
- JavaScript Object Notation (JSON)
- Yaml Ain't Markup Language (YAML)
- Yet Another Next Generation (YANG)

They will understand the value of these data formats in network automation, along with DevOps tools such as Ansible and Git.
